# Client

React frontend for Ajrasakha Hackathon.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Technologies

- React 19
- TypeScript
- Vite
- ESLint
- react-router-dom
- axios
- Tailwind CSS
- react-query(now @tanstack/react-query)
- Zod
